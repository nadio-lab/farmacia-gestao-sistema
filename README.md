# 💊 Gestão de Farmácia - Sistema de Controlo

Um sistema completo de gestão para farmácias, desenvolvido em HTML, CSS, JavaScript e Bootstrap.

## 🎯 Funcionalidades

### 📊 Dashboard
- Estatísticas em tempo real
- Total de medicamentos em stock
- Receita mensal
- Número de clientes
- Alertas de stock baixo
- Últimas vendas

### 💊 Gestão de Medicamentos
- Adicionar/editar/eliminar medicamentos
- Controlo de stock
- Categorização de produtos
- Data de validade
- Preço e código do medicamento

### 🛒 Gestão de Vendas
- Registar novas vendas
- Atualização automática de stock
- Histórico de vendas
- Valor total por transação

### 👥 Gestão de Clientes
- Adicionar/eliminar clientes
- Contacto (email e telefone)
- Data de registo
- Total gasto por cliente

### 🚚 Gestão de Fornecedores
- Registar fornecedores
- Informações de contacto
- Pessoa de contacto
- Histórico

### 📈 Relatórios
- Gerar relatórios por período
- Análise de vendas
- Produtos mais vendidos
- Receita por período

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura
- **Bootstrap 5** - Design responsivo
- **CSS3** - Estilos personalizados
- **JavaScript Vanilla** - Funcionalidades
- **LocalStorage** - Armazenamento de dados

## 📥 Como Usar

1. **Clonar o repositório**
```bash
git clone https://github.com/nadio-lab/farmacia-gestao-sistema.git
cd farmacia-gestao-sistema
```

2. **Abrir no navegador**
Abra o ficheiro `index.html` diretamente no navegador ou use um servidor local:
```bash
python -m http.server 8000
```
Depois aceda a `http://localhost:8000`

## 💾 Armazenamento de Dados

Os dados são armazenados no **LocalStorage** do navegador, o que significa que:
- ✅ Os dados persistem entre sessões
- ✅ Não requer servidor backend
- ✅ Privacidade garantida (dados locais)
- ⚠️ Limite de ~5-10MB por domínio
